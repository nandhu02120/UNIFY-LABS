## Project Description
This is a multi-page HTML5 website for **DevConf 2026**, a fictional technology conference. The project is built using **pure HTML** without any CSS, JavaScript, or frameworks, focusing on proper document structure and semantic HTML tags.

## Pages Included
1.  **index.html**: Homepage with conference details, highlights, and call-to-action.
2.  **about.html**: Information about the conference history, tracks, and target audience.
3.  **speakers.html**: Profiles of keynote speakers.
4.  **schedule.html**: Two-day event schedule with session details.
5.  **sponsors.html**: List of sponsors and sponsorship benefits.

## Technologies Used
- HTML5 (HyperText Markup Language)

## How to Run
1.  Navigate to the project folder.
2.  Open `index.html` in any web browser (Chrome, Firefox, Edge).
3.  Use the navigation links at the top of each page to browse the site.

## Structure
- `/images`: Contains placeholder images for speakers and sponsors.
- `*.html`: The main website pages.
